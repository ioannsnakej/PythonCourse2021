# virtual env
Создание:
python -m venv env
-m — флаг для запуска venv как исполняемого модуля.
env — название виртуального окружения (где будут храниться ваши библиотеки).

Активация:
env/Scripts/activate.ps1 - для Windows;
source env/bin/activate - для Linux и MacOS.

Деактивация: 
deactivate
